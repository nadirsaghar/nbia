NBIA Portal Menu Bar
The NBIA Portal page top menu bar consists of the options described in the following table.
Option
Description
Home
Returns you to the NBIA News home page
Overview
Provides an overview of NBIA and its goals
Download
Displays NBIA software available for download
Collections
Describes existing and future collections.
