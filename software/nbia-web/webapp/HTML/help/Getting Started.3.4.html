Using NBIA Without Logging In
You do not have to log in to access public data in NBIA. Simply begin to use the features on the Welcome page, including search functions.
You can use the search function as a “guest” without logging in. All data that NBIA returns without being logged in is public data. As a guest, you can add data to the data basket. For more information, see Managing the Data Basket.
If you are not logged into the application, some functions are unavailable: saving a search query, viewing image annotations, remote searches and visualizing images using Cedara software. You also cannot download more than 3 gigabytes of data at one time.
If the application is configured to enable the guest account access, the login section will be "hidden" by the login link by default. Additionally, you should be able to see the Search Image link on the home page.
In some cases of local installation, the decision may have been made not to enable public data access without login. If the application is configured to disable the guest account access, the login section is "expanded" by default. If that's the case, there is no way for the user to bypass login and perform tasks.
