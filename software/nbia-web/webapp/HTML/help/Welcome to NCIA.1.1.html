Welcome to the National Biomedical Imaging Archive (NBIA)
The following topics are top-level books in the online help. Each has a variety of subtopics, all of which are designed to introduce you to NCIA and describe concepts you need to know to use NCIA:
•
Getting Started in NBIA
These topics include a brief overview of NCIA, a description of the NBIA home page, and login instructions.
•
Searching for Images
These topics describe the process for searching the NBIA database.
•
Viewing Search Results
These topics describe how to view results from NBIA database searches.
•
Submitting Images
These topics describe the techniques for submitting images to NBIA.
•
NBIA Administration
These topics describe the administration of NBIA, administration roles, responsibilities and permissions.
•
URL References
This appendix contains URLs for sites significant for NBIA
•
NBIA Glossary
The glossary defines acronyms used in the NBIA online help
To learn more about each, browse through the online help, or use the table of contents, index or search features of help. For more information, see Using NBIA Online Help.
